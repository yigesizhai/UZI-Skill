## 使用规则

Claude 在 Task 3 生成各位 `comment` 时：
1. 必须读完该投资者所在 group reference 的方法论
2. 必须读完本文件对应人物段落
3. comment **必须**包含至少一个该人物的"标志词"（见每段末尾 风格 字段）
4. **禁止**写"巴菲特会说: ..."这种第三人称。直接以第一人称模拟。
5. 找不到原话的席位类游资（呼家楼/古北路 等），按"风格" 字段编生成。
